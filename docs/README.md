# Valencia Turismo Sostenible

Este es el directorio de publicación para GitHub Pages. Los archivos aquí están optimizados para servirse a través de GitHub Pages.

## Estructura

```
docs/
├── index.html           # Página principal
├── css/                 # Estilos CSS
├── js/                  # Scripts JavaScript (header.js, footer.js, scripts.js)
├── html/                # Páginas adicionales
├── img/                 # Imágenes del sitio
└── .nojekyll           # Archivo para deshabilitar Jekyll en GitHub Pages
```

## Cómo usar

1. Este sitio está configurado para servirse desde la rama `main`, carpeta `docs/`
2. Ve a tu repositorio en GitHub → Settings → Pages
3. Selecciona "Deploy from a branch"
4. Rama: `main`
5. Carpeta: `docs`
6. Haz clic en "Save"

Tu sitio estará disponible en: `https://manuelmag1.github.io/ValenciaTurismo/`

## Característocas

- ✅ Diseño responsive con Tailwind CSS
- ✅ Navegación dinámica con JavaScript
- ✅ 16 páginas sobre sostenibilidad en Valencia
- ✅ Galerías de imágenes optimizadas
- ✅ Modo oscuro soportado

## Desarrollo

Para cambios locales, edita los archivos en la raíz del proyecto (carpeta `html/`, `css/`, `js/`, `img/`) y luego copia los cambios a esta carpeta `docs/` o usa un script de build.

## 📁 Estructura del repositorio

```
│
├── docs/                    # ← Carpeta publicada en GitHub Pages
│   ├── index.html
│   ├── *.html               # Todas las páginas (17 en total)
│   ├── css/

│
├── html/                    # Archivos fuente locales (no en GitHub)
├── css/                     # Estilos locales
├── js/                      # Scripts locales
├── img/                     # Imágenes locales
├── .gitignore               # Ignora archivos fuente
└── README.md
```

## 💻 Desarrollo local

**Archivos fuente** (`html/`, `css/`, `js/`, `img/`) se mantienen localmente pero **no se suben a GitHub**.

Cuando hagas cambios:
1. Edita en las carpetas locales (`html/`, `css/`, etc.)
2. Copia los cambios a la carpeta `docs/`
3. Hace `git add docs/` y `git push`
